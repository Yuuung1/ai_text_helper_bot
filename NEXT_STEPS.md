# Next Steps

## Текущая стабильная версия

Проект содержит:

- консольную утилиту `main.py`;
- Telegram-бота `bot.py`;
- общую логику анализа текста в `text_tools.py`;
- тесты `test_text_tools.py` и `test_bot.py`;
- зависимости в `requirements.txt`;
- переменные окружения через `.env` и `.env.example`.

## Что уже умеет бот

- `/start` — приветствие;
- `/help` — инструкция;
- обычный текст — отчёт по тексту;
- обработка пустого текста;
- ограничение длины отчёта для Telegram.

## Следующий большой этап

Добавить AI-функцию `/summary`.

План:

1. Выбрать бесплатный AI API-провайдер.
2. Скорее всего начать с Gemini API через Google AI Studio.
3. Добавить в `.env.example`:
   - `AI_PROVIDER=gemini`
   - `GEMINI_API_KEY=put_your_gemini_api_key_here`
   - `AI_MODEL=...`
4. Создать `ai_client.py`.
5. Добавить функцию `summarize_text(text: str) -> str`.
6. Добавить команду `/summary` в `bot.py`.
7. Проверить сценарий:
   - `/summary длинный текст`
   - бот возвращает краткое резюме.
8. Добавить обработку ошибок:
   - нет ключа;
   - пустой текст;
   - ошибка API;
   - слишком длинный ответ.
9. Обновить README.
10. Добавить тесты там, где возможно.

## Важный принцип

`bot.py` не должен содержать AI-логику напрямую.

Правильная схема:

Telegram → bot.py → ai_client.py → AI API → bot.py → Telegram
