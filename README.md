# AI Text Helper Bot — Text Report Cleaner

Небольшая Python-утилита для анализа текстовых файлов.

Программа читает `.txt` файл, очищает текст от лишних пробелов, считает статистику и сохраняет отчёт в отдельный файл.

## Возможности

- чтение текстового файла;
- удаление лишних пробелов и пустых строк;
- подсчёт строк, слов и символов;
- подсчёт уникальных слов;
- поиск самого длинного слова;
- вывод топ-10 самых частых слов;
- сохранение результата в отчёт;
- запуск с аргументами командной строки.

## Пример запуска

Запуск со стандартным файлом отчёта:

```bash
python main.py sample.txt
```

Запуск с указанием файла отчёта:

```bash
python main.py sample.txt custom_report.txt
```

Запуск с другим входным файлом:

```bash
python main.py another_text.txt another_report.txt
```

## Пример результата

## Telegram-бот

Проект также содержит Telegram-интерфейс для анализа текста.

Бот принимает текстовое сообщение, очищает его и возвращает отчёт со статистикой.

### Настройка токена

Создайте файл `.env` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
```

Пример структуры настроек есть в файле `.env.example`.

### Запуск бота

```bash
python bot.py
```

После запуска откройте бота в Telegram и отправьте:

```text
/start
```

или любой текст для анализа.

### Проверенные сценарии

- `/start` — приветствие;
- `/help` — инструкция;
- обычное текстовое сообщение — отчёт по тексту.

## Установка зависимостей

После клонирования проекта создайте виртуальное окружение и установите зависимости:

```bash
python -m venv .venv
pip install -r requirements.txt
```

Для запуска Telegram-бота также нужен файл `.env` с токеном:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
```
